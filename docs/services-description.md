# 🛠️ GeoTarget Pro — Services & Tools Description

# خدمات اور ٹولز کی تفصیل | وصف الخدمات والأدوات

---

## 1. React.js (Frontend Framework)

**🇺🇸 English:** React.js is a JavaScript library for building user interfaces. It powers GeoTarget Pro's interactive dashboard, maps, and all UI components with fast, reactive rendering.

**🇵🇰 اردو:** React.js ایک JavaScript لائبریری ہے جو یوزر انٹرفیسز بنانے کے لیے استعمال ہوتی ہے۔ یہ GeoTarget Pro کے انٹرایکٹیو ڈیش بورڈ، میپس اور تمام UI کمپوننٹس کو تیز رینڈرنگ فراہم کرتی ہے۔

**🇸🇦 العربية:** React.js هي مكتبة JavaScript لبناء واجهات المستخدم. تُشغّل لوحة التحكم التفاعلية والخرائط وجميع مكونات الواجهة بعرض سريع.

---

## 2. Vite (Build Tool)

**🇺🇸** Vite is a next-generation frontend build tool that provides lightning-fast development server and optimized production builds.

**🇵🇰** Vite ایک جدید فرنٹ اینڈ بلڈ ٹول ہے جو انتہائی تیز ڈیولپمنٹ سرور اور آپٹیمائزڈ پروڈکشن بلڈز فراہم کرتا ہے۔

**🇸🇦** Vite أداة بناء واجهات أمامية من الجيل التالي توفر خادم تطوير سريع وبناء إنتاج محسّن.

---

## 3. Tailwind CSS (Styling)

**🇺🇸** Utility-first CSS framework for rapidly building custom designs. Provides responsive, dark-mode enabled styling for the entire application.

**🇵🇰** یوٹیلیٹی فرسٹ CSS فریم ورک جو تیزی سے کسٹم ڈیزائنز بنانے کے لیے استعمال ہوتا ہے۔ پوری ایپلیکیشن کے لیے ریسپانسیو اور ڈارک موڈ اسٹائلنگ فراہم کرتا ہے۔

**🇸🇦** إطار عمل CSS قائم على الأدوات لبناء تصاميم مخصصة بسرعة مع دعم الوضع المظلم والتصميم المتجاوب.

---

## 4. Leaflet.js (Map Library)

**🇺🇸** Open-source JavaScript library for interactive maps. Used for displaying geofences, drawing boundaries, and location visualization — free with no API key required.

**🇵🇰** انٹرایکٹیو میپس کے لیے اوپن سورس JavaScript لائبریری۔ جیو فینسز دکھانے، باؤنڈریز بنانے اور لوکیشن ویژولائزیشن کے لیے استعمال ہوتی ہے — مفت اور بغیر API کلید۔

**🇸🇦** مكتبة JavaScript مفتوحة المصدر للخرائط التفاعلية. تُستخدم لعرض السياجات الجغرافية ورسم الحدود — مجانية بدون مفتاح API.

---

## 5. Node.js + Express.js (Backend)

**🇺🇸** Node.js is a JavaScript runtime for server-side development. Express.js is a minimal web framework that handles API routes, authentication, and geofence logic.

**🇵🇰** Node.js سرور سائیڈ ڈیولپمنٹ کے لیے JavaScript رن ٹائم ہے۔ Express.js ایک ویب فریم ورک ہے جو API روٹس، آتھنٹیکیشن اور جیو فینس لاجک کو ہینڈل کرتا ہے۔

**🇸🇦** Node.js بيئة تشغيل JavaScript للتطوير من جانب الخادم. Express.js إطار ويب يتعامل مع مسارات API والمصادقة ومنطق السياج الجغرافي.

---

## 6. MongoDB (Database)

**🇺🇸** NoSQL document database for storing geofences (GeoJSON), user accounts, audience data, and analytics. Supports geospatial queries natively.

**🇵🇰** NoSQL ڈاکیومنٹ ڈیٹابیس جو جیو فینسز (GeoJSON)، یوزر اکاؤنٹس، آڈئینس ڈیٹا اور اینالیٹکس سٹور کرتا ہے۔ جیو سپیشل کیوریز کی مقامی سپورٹ۔

**🇸🇦** قاعدة بيانات مستندات NoSQL لتخزين السياجات الجغرافية وحسابات المستخدمين وبيانات الجمهور. يدعم الاستعلامات الجغرافية.

---

## 7. JWT (Authentication)

**🇺🇸** JSON Web Tokens for secure, stateless authentication. Users receive encrypted tokens upon login that authorize API requests.

**🇵🇰** محفوظ، اسٹیٹ لیس آتھنٹیکیشن کے لیے JSON ویب ٹوکنز۔ لاگ ان پر یوزرز کو انکرپٹڈ ٹوکنز ملتے ہیں جو API ریکوئسٹس کو آتھورائز کرتے ہیں۔

**🇸🇦** رموز ويب JSON للمصادقة الآمنة. يتلقى المستخدمون رموزاً مشفرة عند تسجيل الدخول تصرح بطلبات API.

---

## 8. Chart.js (Analytics)

**🇺🇸** JavaScript charting library for creating beautiful, responsive charts. Used for visitor analytics, dwell time graphs, and performance dashboards.

**🇵🇰** JavaScript چارٹنگ لائبریری جو خوبصورت، ریسپانسیو چارٹس بنانے کے لیے استعمال ہوتی ہے۔ وزیٹر اینالیٹکس اور پرفارمنس ڈیش بورڈز کے لیے۔

**🇸🇦** مكتبة JavaScript للرسوم البيانية لإنشاء مخططات جميلة ومتجاوبة. تُستخدم لتحليلات الزوار ولوحات الأداء.

---

## 9. Bcrypt.js (Security)

**🇺🇸** Password hashing library that securely encrypts user passwords before storing them in the database. Uses salt rounds for additional security.

**🇵🇰** پاس ورڈ ہیشنگ لائبریری جو یوزر پاس ورڈز کو ڈیٹابیس میں سٹور کرنے سے پہلے محفوظ طریقے سے انکرپٹ کرتی ہے۔

**🇸🇦** مكتبة تجزئة كلمات المرور تُشفّر كلمات مرور المستخدمين بشكل آمن قبل تخزينها في قاعدة البيانات.

---

## 10. Vercel / Railway / AWS (Hosting)

**🇺🇸** Cloud hosting platforms for deploying the application. Vercel hosts the frontend, Railway/AWS hosts the backend API, and MongoDB Atlas provides managed database.

**🇵🇰** ایپلیکیشن ڈیپلائی کرنے کے لیے کلاؤڈ ہوسٹنگ پلیٹ فارمز۔ Vercel فرنٹ اینڈ ہوسٹ کرتا ہے، Railway/AWS بیک اینڈ ہوسٹ کرتا ہے، اور MongoDB Atlas ڈیٹابیس فراہم کرتا ہے۔

**🇸🇦** منصات استضافة سحابية لنشر التطبيق. Vercel يستضيف الواجهة الأمامية، Railway/AWS يستضيف الخادم الخلفي، وMongoDB Atlas يوفر قاعدة بيانات مُدارة.
